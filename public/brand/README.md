# Anderson Cleaning – Brand Asset Package

This package contains the complete, organized, and production‑ready brand assets for Anderson Cleaning.  
It includes all logos, icons, and favicons in both color and white (for dark mode), fully optimized and structured for use in any modern website or application.

## What This Package Contains

### 1. Color Logo Assets (Light Mode)
These are the primary Anderson Cleaning logos and icons:
- Full logo (2000px)
- Icon‑only logos (256px, 512px, 1024px, 2000px)
- Fully optimized favicon set (16px – 512px)

### 2. White Logo Assets (Dark Mode)
These are the white versions of the same assets, designed to look clean and crisp on dark backgrounds or when the website is in dark mode.

### 3. Organized Folder Structure
The assets are placed in a clean, Apple‑style structure:
- `brand/color/` – All light‑mode icons and logos  
- `brand/white/` – All dark‑mode icons and logos  

This keeps your project tidy and makes it easy for developers or AI coding assistants to find the exact file needed.

## Purpose of This Package
This brand package ensures your visual identity:
- Looks clean, premium, and consistent  
- Works flawlessly in light and dark modes  
- Meets modern standards for browsers, apps, and devices  
- Stays organized for long‑term development and scaling  

## Placement in Your Project
You should place the `brand/` folder inside your website’s `/public/` directory:
```
/public/brand/color  
/public/brand/white  
```

This makes all assets accessible to your site without needing import statements.
